Youtube: [API Structure Your Nodejs REST API for beginner to Advanced](https://youtu.be/i4Pr81apfnU)

# How to use

### clone git

```
> git clone https://github.com/anonystick/structure-api-mvc-express-nodejs.git your-project
> cd your-project
> npm i
> npm run dev
```
